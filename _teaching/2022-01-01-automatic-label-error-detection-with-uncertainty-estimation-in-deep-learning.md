Supervised a student project (internship) on automatic detection of annotation errors in LiDAR perception datasets using uncertainty estimation for deep learning models.

Main focus areas:

- literature review on uncertainty estimation and label error detection
- implementation of an automatic label error detection tool in TensorFlow or PyTorch
- design and evaluation of experiments with different uncertainty estimation methods
- refinement of the detection pipeline with additional heuristics

The project targeted reliable training data curation for semantic segmentation in autonomous driving.
